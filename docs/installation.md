---

## Enabling The Plugin

If obtained through Fab : 

- The plugin should be automatically added to the project

- Check and enable the plugin.


If Obtained manually, through Github & Itch.io : 

- Download the zip file.

- Create a folder in your project file called `Plugins`.

- Extract and copy to `Plugins`

- (optional but recommened) Do a hard recompile (Delete Binaries, Intermediate, Saved, etc.)

- Open Project amd ensure the project is Enabled.


All Set and ready to go!
